    Weather Drink

    Purpose:
    Create a web-based application that utilizes multiple APIs to solve an issue.

    Description:
    At team based project that upon visiting our page the user will be given a suggestion for a cocktail and a recipe. This recommendation will be based off the weather of the current users location. Looking at various weather statistics such as temperature, rain, snow or sunshine we will attempt to match a drink with ingredients that reflect those conditions.

    Technologies Used:
    -HTML -CSS -Javascript -Jquery -Foundation -Slick.JS -WeatherIcons.io
    APIs: -OpenWeatherMap -TheCocktailDB -Google Maps.

    Future Improvements:
    Add additional weather conditions and improve style.
